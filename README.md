# English Memory Game

Откройте [index.html](/Users/worldmac/Documents/Слова на англ/Слова на англ 2/index.html) в браузере, и тренажер сразу заработает без установки зависимостей.

Что внутри:
- `Карточки памяти`: английское слово показывается без русского перевода, перевод открывается только по кнопке.
- `Пиши по-английски`: видно русское слово, а ответ нужно ввести на английском.
- `Слушать`: есть в обоих режимах, озвучка идет через встроенный голос браузера.

Если захотите поправить слова или добавить новые, редактируйте [words.js](/Users/worldmac/Documents/Слова на англ/Слова на англ 2/words.js).

Примечание: слова я аккуратно восстановил по вашим фото, но OCR на нескольких строчках был неидеальным, поэтому при желании список легко подправить в одном месте.
