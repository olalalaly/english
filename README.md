# English Memory Game

Откройте [index.html](/Users/worldmac/Documents/Слова на англ/Слова на англ 2/index.html) в браузере, и тренажер сразу заработает без установки зависимостей.

Что внутри:
- `Группы по 12 слов`: можно учить слова маленькими блоками.
- `Карточки памяти`: английское слово показывается без русского перевода, перевод открывается только по кнопке.
- `Русский → English`: письменная тренировка перевода на английский.
- `English → Русский`: письменная тренировка перевода на русский.
- `Рейтинг 0-5`: у каждого слова есть свой уровень знания, который можно менять вручную и который обновляется после ответов.
- `Слушать`: есть во всех режимах, озвучка идет через встроенный голос браузера.

Если захотите поправить слова или добавить новые, редактируйте [words.js](/Users/worldmac/Documents/Слова на англ/Слова на англ 2/words.js).

Примечание: слова я аккуратно восстановил по вашим фото, но на нескольких строках почерк был сложный, поэтому список всегда можно быстро подправить в одном месте.
